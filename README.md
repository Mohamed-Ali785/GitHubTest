# Navbar Project

A simple and responsive **navigation bar (navbar)** built with HTML, CSS, and optionally JavaScript. This project demonstrates how to create a clean, modern, and functional navigation menu for websites.

---

## Features

- Responsive design (works on desktop, tablet, and mobile)
- Dropdown menus for sub-navigation items
- Smooth hover effects
- Easy to customize colors, fonts, and layout

---


## Technologies Used

- HTML4
- CSS
